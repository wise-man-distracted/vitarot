# App de tarô
App feito para treinar conceitos de front-end e iteração com arrays e objetos. Completamente funcional, porém ainda em fase de desenvolvimento.


## Teconologias usadas
Vite
Tailwind
Bootstrap
HTML
JS
CSS

## Como instalar
0. Clone o repositório
1. `npm i`
2. `npm run dev`
3. Acesse: http://localhost:3000
